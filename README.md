# Brain Training App

A comprehensive cognitive training application designed to improve mental focus, memory, and processing speed through scientifically-backed exercises.

## Features

- **Multiple Training Modules** - Various cognitive exercises targeting different mental skills
- **Progress Tracking** - Monitor your improvement over time with detailed analytics
- **Adaptive Difficulty** - Exercises adjust to your skill level
- **Daily Challenges** - Stay motivated with curated daily cognitive tasks
- **Responsive Design** - Seamless experience across desktop and mobile devices
- **Dark Mode Support** - Easy on the eyes during extended sessions

## Tech Stack

- **Frontend:** JavaScript, React
- **Styling:** CSS/Tailwind
- **State Management:** React Context or Redux
- **Data Storage:** LocalStorage for user progress

## Getting Started

### Prerequisites
- Node.js 14+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/clydeagnew-max/clyde-brain-training-app.git
cd clyde-brain-training-app

# Install dependencies
npm install

# Start the development server
npm start
```

The app will open at `http://localhost:3000`

## Usage

1. **Create Account** - Sign up or log in
2. **Select Exercise** - Choose from available training modules
3. **Train** - Complete the cognitive exercises
4. **Review Progress** - Check your performance metrics
5. **Daily Streak** - Maintain consistency for better results

## Available Exercises

- Memory Matching
- Sequence Recognition
- Quick Calculation
- Visual Perception
- Pattern Recognition
